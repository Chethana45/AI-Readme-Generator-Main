## Backend Setup

```bash
cd server

npm install
```

Create a `.env` file:

```env
PORT=5000
GEMINI_API_KEY=YOUR_API_KEY
```

Start Backend:

```bash
npm start
```

## Frontend Setup

```bash
cd client

npm install

npm run dev
```

Application runs on:

```bash
Frontend : http://localhost:5173

Backend : http://localhost:5000
```

## рџљЂ How It Works

1. User enters a GitHub repository URL.
2. GitHub API fetches repository information.
3. Repository structure is analyzed.
4. Gemini AI generates a professional README.
5. Generated README is displayed in preview.
6. User can download README.md instantly.
